# Defensive Situations — Upper Little League (9–12)

A reference guide for all nine positions covering the most common defensive situations. Use this for player education, quizzes, and pre-season review.

---

## How to Use This Guide

- Each situation lists what **every position** should do.
- Items marked **⚠️ COACH DECISION** indicate plays where teams commonly do it differently. Coaches should define their preference at the start of the season so players learn one consistent system.
- **Communication rules always apply:** "I got it!" / "You got it!" Outfielders override infielders. CF overrides all outfielders. On any relay, the cut-off listens for "Cut!" (intercept and throw home) or "Let it go!" (ball goes through).

---

## General Priority Rules (All Situations)

| Priority | Rule |
|----------|------|
| Highest | Outfielders call off all infielders on fly balls |
| | CF calls off all other outfielders |
| | SS calls off all other infielders |
| Lowest | Catcher fields pop-ups only when no one else can reach |

## Section 3: Fly Balls — Runner on 2B (Tag-Up Situations)

The runner on 2B may tag and attempt to score. The throw goes home.

### 3.1 Fly Ball to Left Field — Runner on 2B

| Position | Responsibility |
|----------|----------------|
| P | Sprint to back up home plate |
| C | Set up at home; prepare to receive throw and apply tag |
| 1B | Cover 1B |
| 2B | Cover 2B |
| SS | Move into relay between LF and home |
| 3B | Cover 3B; remind runner of tag-up timing |
| LF | Catch; hard throw to SS relay toward home |
| CF | Back up LF |
| RF | Shift toward infield |

> ⚠️ **COACH DECISION — Cut-off to home plate:** On throws from LF/CF, who is the relay/cut-off? Common options:
> - **SS** (most common — already moving toward LF)
> - **Pitcher** (shorter relay, simpler for young players)
> - **3B** (used by some teams when 3B is not needed at the bag)
>
> The **catcher** directs the cut-off: yells "Cut! Throw to ___" or "Let it go!"

## Section 5: Ground Balls

### 5.1 Ground Ball — No Runners On

All infielders: field your position and make the routine out at 1B.

| Position | Responsibility |
|----------|----------------|
| P | Cover 1B if 1B fields the ball; back up 1B on long throws from 3B/SS side |
| C | Cover home plate |
| 1B | Field ball or cover 1B bag; receive throw |
| 2B | Field ball or cover 2B area |
| SS | Field ball or cover 2B area as backup |
| 3B | Field ball; throw to 1B |
| LF | Back up 3B/SS on hard-hit balls through the left side |
| CF | Back up 2B/SS area |
| RF | Back up 1B/2B area |

---

### 5.2 Ground Ball — Runner on 1B (Double Play Situation)

Force at 2B; relay to 1B for double play.

| Position | Responsibility |
|----------|----------------|
| P | Field if hit to P; throw to 2B; cover 1B if 1B leaves bag |
| C | Cover home |
| 1B | Cover 1B; receive final throw for double play |
| 2B | Cover 2B — receive first throw, pivot, relay to 1B |
| SS | Cover 2B OR field ground ball and throw to 2B |
| 3B | Field ball if to 3B; throw to 2B for double play |
| LF | Back up 3B area |
| CF | Back up 2B area |
| RF | Back up 1B area |

> ⚠️ **COACH DECISION — Who covers 2B on the double play?**
> - **Traditional:** SS covers 2B on balls hit to the right side (1B, 2B, P); 2B covers 2B on balls hit to the left side (3B, SS, P left of mound)
> - **Simplified (common for youth):** SS always covers 2B on all double play balls — easier for young players to remember
>
> Choose one system. Inconsistency here causes collisions or uncovered bags.

---

### 5.3 Ground Ball — Runners on 1B & 2B

Force at 3B, 2B, and 1B. Multiple double play options.

| Position | Responsibility |
|----------|----------------|
| P | Field if to P; throw to 3B or 2B per team's system |
| C | Cover home |
| 1B | Cover 1B; receive final throw |
| 2B | Cover 2B; pivot and relay to 1B |
| SS | Cover 3B OR field and throw to 3B/2B |
| 3B | Field ball if to 3B; throw to 2B for double play; OR SS covers 3B and 3B charges the ball |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

> ⚠️ **COACH DECISION — Double play sequence with 1B & 2B:** Common options:
> - **3B → 2B → 1B** (standard, hardest for runner to beat)
> - **3B → home → 1B** (if bases are loaded; see 5.4)
> - **Simplify to just get one out** (depending on score/inning)

---

### 5.4 Ground Ball — Bases Loaded

Force at every base. Multiple double play options.

| Position | Responsibility |
|----------|----------------|
| P | Field if to P; throw home per team's system |
| C | Set up at home; receive throw, relay to 1B for DP |
| 1B | Cover 1B; field if to 1B, throw home |
| 2B | Cover 2B |
| SS | Cover 3B or field ball |
| 3B | Field ball; throw home or to 2B |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

> ⚠️ **COACH DECISION — Priority with bases loaded:** Define the default:
> - **Throw home first, relay to 1B** (double play, no run scores)
> - **Throw to 3B, relay to 1B** (double play, run scores)
> - **Get the sure out at 1B** (one out, runners advance — conservative)
>
> This decision should also factor in outs in the inning and game score.

## Section 7: Bunt Defense

### 7.1 Bunt Defense — Runner on 1B

Goal: get the lead runner at 2B, or settle for the out at 1B.

| Position | Responsibility |
|----------|----------------|
| P | Charge and field; throw to 2B (lead runner) or 1B |
| C | Pop up and direct traffic; field if the bunt is right at home |
| 1B | Charge; if fielding, throw to 2B; if not, cover 1B |
| 2B | Cover 1B (since 1B is charging) |
| SS | Cover 2B |
| 3B | Charge aggressively from bag |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

> ⚠️ **COACH DECISION — Bunt rotation with runner on 1B:**
> - **Option A (aggressive):** P + 1B + 3B all crash — 2B covers 1B, SS covers 2B
> - **Option B (conservative):** P + 3B charge — 1B stays near bag, 2B covers 2B area
> - **Option C:** P + 3B + C (if bunt is popped near plate)
>
> Also define: when the bunt is fielded, is the default throw **always to 1B** (safe out) or **always try to get the lead runner at 2B**? Many youth coaches simplify to "just get the out at 1B."

---

### 7.2 Bunt Defense — Runners on 1B & 2B (Sacrifice)

Goal: force out at 3B, or at minimum get an out at 1B.

| Position | Responsibility |
|----------|----------------|
| P | Charge and field; throw to 3B (force) if close enough; otherwise 1B |
| C | Direct traffic; field if bunt dies near plate |
| 1B | Charge; throw to 3B if fielding; if not, cover 1B |
| 2B | Cover 1B |
| SS | Sprint to cover 3B |
| 3B | Charge; if NOT fielding, retreat immediately to cover 3B |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

> ⚠️ **COACH DECISION — Bunt rotation with 1B & 2B:** Same options as 7.1 apply, plus:
> - Is the primary goal to **force at 3B** (aggressive) or **take the sure out at 1B** (conservative)?
> - Who has priority calling the fielder off the bunt — **C** or **P**? Define who makes the "I got it!" call.

## Section 8: Stolen Base Defense

### 8.1 Steal of 2B

| Position | Responsibility |
|----------|----------------|
| P | Deliver pitch; stay alert; do not obstruct throw |
| C | Receive pitch cleanly; quick accurate throw to 2B |
| 1B | Hold runner at 1B until pitch is delivered |
| 2B | Break toward 2B to cover — or hold (based on assignment) |
| SS | Break toward 2B to cover — or hold (based on assignment) |
| 3B | Stay near 3B |
| LF | Sprint to back up 2B from left-center |
| CF | Back up 2B from center field |
| RF | Drift toward 2B area |

> ⚠️ **COACH DECISION — Who covers 2B on a steal?**
> - **Traditional:** SS covers on a right-handed batter (has better angle); 2B covers on a left-handed batter
> - **Simplified (recommended for youth):** SS always covers 2B on steals — one less thing for players to think about
> - **Signal system:** Some coaches use hand signals from dugout to tell SS and 2B who covers on each pitch
>
> 2B and SS must communicate before every pitch when a runner is on 1B.

## Section 9: First & Third Steal Defense

Runner on 1B steals 2B while runner on 3B watches for an opportunity to score. One of the most complex situations at this level.

> ⚠️ **COACH DECISION — First & Third defense system:** This is the highest-variation situation in youth baseball. Define ONE system your players practice:
>
> | Option | What Happens |
> |--------|-------------|
> | **A — Throw through:** C throws full speed to 2B; SS/2B must cut it if runner breaks from 3B | High risk, can stop a good steal |
> | **B — Fake and look:** C pump fakes toward 2B, then throws to 3B to catch runner leaning | Requires good timing; effective if runner on 3B bites |
> | **C — Throw to P (cut):** C throws to P standing on the mound; P reads runner on 3B and throws home or to 2B | Simple and safe for youth |
> | **D — Throw to SS (cut):** C throws to SS who cuts it if runner breaks; SS either throws home or lets it go to 2B | Requires SS to make quick read |
>
> Whatever system you choose, players must know their roles: who cuts, who covers 2B, who covers home.

| Position | Responsibility (adjust per system above) |
|----------|------------------------------------------|
| P | Per system: serve as cut-off OR retreat and be ready to cover home |
| C | Receive pitch; execute throw per team's system |
| 1B | Cover 1B |
| 2B | Cover 2B (receive throw or trail the play) |
| SS | Cover 2B OR serve as cut-off per system |
| 3B | Stay near 3B; watch runner on 3B; communicate |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

## Section 11: Infield Pop-Ups

### 11.1 Priority Rules — Infield Pop-Up

| Priority | Position | Rule |
|----------|----------|------|
| Highest | Outfielders | Call off everyone; running in on a pop-up have the best angle |
| | SS | Calls off all other infielders |
| | 2B | Calls off 1B, 3B, and P |
| | 1B / 3B | Yields to SS and 2B |
| | P | Yields to all infielders |
| Lowest | C | Fields pop-up only if no one else can reach; everyone else calls C off |

**On foul pop-ups near dugout or stands:** C and corner infielders communicate. The fielder who calls for it owns it.

## Section 12: Rundowns

### 12.1 Rundown — Any Bases

> ⚠️ **COACH DECISION — Rundown mechanics:** Teams handle rundowns differently. Define your approach:
> - **Goal:** Always run the runner *back toward the base they came from* (prevent advancement)
> - **Throws:** Minimize to 1–2 throws maximum
> - **Who runs the rundown:** The fielder holding the ball runs hard at the runner; the other end receives the throw and applies the tag

| Position | General Responsibility |
|----------|-----------------------|
| Fielder with ball | Run hard at the runner; throw only when runner commits to the far base |
| Receiving fielder | Set up inside the baseline to receive throw and apply tag |
| All other fielders | Back up one of the two bases involved in the rundown |
| P | Fill in any uncovered bag |
| C | Cover home if rundown is between 3B and home |

**Critical rule: Never let the runner advance a base during a rundown.**

## Section 13: Outfield Throwing Priority Reference

### 13.1 Primary Throw Target by Base Situation

The outfielder's job is to throw **one base ahead of the lead runner**. Before every pitch, each outfielder should know exactly where their throw is going if the ball is hit to them.

| Base Situation | Outfielder's Default Primary Target |
|----------------|-------------------------------------|
| No runners | 2B (batter rounding 1B; hit cutoff if ball is deep) |
| Runner on 1B | 3B (runner advancing to 3B is the lead threat) |
| Runner on 2B | Home (runner scoring is the lead threat) |
| Runner on 3B | Home |
| Runners on 1B & 2B | Home (lead runner from 2B) |
| Runners on 1B & 3B | Home (runner on 3B) |
| Runners on 2B & 3B | Home |
| Bases loaded | Home |

> **Reading the play — when to deviate:** If the lead runner clearly stops (e.g., runner on 1B stays at 2B on a routine single), redirect to the next threat. The outfielder must always **look** before throwing. Don't airmailed a ball to an empty base.

> ⚠️ **COACH DECISION — Who reads the throw target?**
> - **Outfielder reads independently:** Outfielder sees the base situation and throws to the right base without being told. Higher skill ceiling; relies on outfielder awareness.
> - **Always throw to the cutoff:** Outfielder throws hard to the relay/cutoff every time; the cutoff reads the play and either cuts it or lets it go (directed by C). Simpler for younger players; reduces mental load on outfielder.
>
> Define one system. Mixed signals between outfielder and cutoff cost runs.

### 13.2 Runner on 1B — Throw Target by Field Position

Because the runner on 1B is running with the pitch, where the ball is hit affects how far they advance.

| Ball Hit To | Runner on 1B Likely Stops At | Outfielder's Target |
|-------------|------------------------------|---------------------|
| LF (single) | 2B or 3B depending on speed/jump | 3B (relay) |
| CF (single) | 3B | 3B (relay) |
| RF (single) | 2B (running away from throw) | 2B or 3B |
| Deep extra-base hit (any field) | 3B or home | Home (relay chain) |

## Section 14: Ground Balls — Runner on 3B

### 14.1 Ground Ball — Runner on 3B Only, Fewer Than 2 Outs ("Look Runner Back")

The infielder fields the ball and **briefly looks at the runner on 3B** to freeze them before throwing to 1B. If the runner breaks for home, the infielder throws home instead.

| Position | Responsibility |
|----------|----------------|
| P | Field if hit to P; look to 3B; throw to 1B or home based on runner's decision |
| C | Set up at home plate; be ready to receive throw and apply tag |
| 1B | Cover 1B bag; receive throw for the out |
| 2B | Cover 2B |
| SS | Back up 3B; be ready if throw goes to 3B |
| 3B | Field if hit to 3B; look to runner; throw to 1B or home; if not fielding, hold near bag |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

**Key rules:**
- Look is a **quick glance**, not a long hesitation — don't let the batter-runner reach safely because you stared too long
- If the runner commits to home, throw home; don't fake and throw to 1B (runner scores anyway and you might not get the out)
- On a slow roller or bunt-like ball, there may be **no time to look** — just throw to 1B

> ⚠️ **COACH DECISION — Infield depth with runner on 3B, fewer than 2 outs:**
> - **Normal depth:** Reduces gaps; harder to throw runner out at home if they break early
> - **Drawn in (halfway):** Infielders move in several steps; can cut down the runner at home but opens holes for base hits through
> - **All the way in:** Rarely used at youth level; used in late-inning, close-game situations to cut down any runner at home
>
> Depth changes where every infielder stands before the pitch. Practice this alignment.

> ⚠️ **COACH DECISION — Who makes the "look" call?**
> - **Fielder decides independently:** Reads the runner's break and makes own choice (throw to 1B or home)
> - **Catcher directs:** C reads the runner and yells "Home!" or "One!" to tell fielder where to throw — reduces decision burden on fielder but requires loud, fast communication

---

### 14.2 Ground Ball — Runner on 3B Only, 2 Outs

With 2 outs, the runner goes on contact regardless. No need to look the runner back — they are already running.

| Position | Responsibility |
|----------|----------------|
| P | Field if hit to P; throw to 1B — get the out |
| C | Cover home (runner will score; focus is on getting the third out) |
| 1B | Cover 1B; receive throw |
| 2B | Cover 2B |
| SS | Cover SS area |
| 3B | Field if hit to 3B; throw to 1B |
| LF | Back up 3B area |
| CF | Back up 2B area |
| RF | Back up 1B area |

**Key rule:** The run scores if the infielder gets the out — that is acceptable. Focus entirely on getting the third out cleanly.

---

### 14.3 Infield Depth Reference

| Depth | When Used | Effect |
|-------|-----------|--------|
| **Normal** | Default; no runners or runner not threatening | Good range on all ground balls |
| **Double play depth** | Runner on 1B, fewer than 2 outs | SS and 2B cheat toward 2B; 1B and 3B hold normal; quicker pivot for DP |
| **Drawn in (halfway)** | Runner on 3B, fewer than 2 outs | Can cut down runner at home; leaves gaps for hits |
| **All the way in** | Late inning, close game, runner on 3B | Best chance to cut down any ground ball runner; high risk of balls getting through |

## Section 15: Ground Balls — Additional Runner Combinations

### 15.1 Ground Ball — Runner on 2B Only, Fewer Than 2 Outs

Runner on 2B may try to score on a ground ball through the infield. The infielder should be aware of the runner but prioritize the out at 1B.

| Position | Responsibility |
|----------|----------------|
| P | Field if to P; look back at runner on 2B briefly; throw to 1B |
| C | Cover home; be ready if runner breaks early on a slow roller |
| 1B | Cover 1B; receive throw |
| 2B | Cover 2B after fielding (SS covers if 2B fields) |
| SS | Field ball or cover 2B; call out runner's position |
| 3B | Field if to 3B; look at runner; throw to 1B or home |
| LF | Back up toward 3B |
| CF | Back up 2B area |
| RF | Back up 1B |

**Key rule:** A runner on 2B generally should not score on a routine infield ground ball. If they try, the infielder may have a play at home. On a ball hit directly at an infielder, throw to 1B — the runner won't score. On a slow roller or ball up the middle where the runner had a good jump, C should be ready.

---

### 15.2 Ground Ball — Runners on 1B & 3B, Fewer Than 2 Outs

Double play is available at 2B and 1B, but the runner on 3B will try to score if the fielder throws to 2B. The infielder must quickly decide: look the runner at 3B back, then go for the double play.

| Position | Responsibility |
|----------|----------------|
| P | Field if to P; look at runner on 3B; go for DP at 2B (or throw home if runner breaks) |
| C | Cover home; direct play from behind; call "Home!" if runner breaks |
| 1B | Cover 1B; receive final throw for double play |
| 2B | Cover 2B; receive throw and pivot for relay to 1B |
| SS | Cover 2B OR field and throw to 2B; call position of runner on 3B |
| 3B | Field if to 3B; look at runner on 3B; throw to 2B for DP |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

> ⚠️ **COACH DECISION — Runner on 3B, going for double play:** When the fielder throws to 2B, the runner on 3B often scores. Define your team's priority:
> - **Always go for the DP** — accept that the runner scores; get 2 outs
> - **Hold runner, get sure out at 1B** — if runner on 3B is dancing and threatening, just throw to 1B and keep the run off the board

---

### 15.3 Ground Ball — Runners on 2B & 3B, Fewer Than 2 Outs

Force play only exists at 1B. Both runners will try to advance on contact.

| Position | Responsibility |
|----------|----------------|
| P | Field if to P; look at runner on 3B; throw to 1B |
| C | Cover home; be ready if runner on 3B breaks early |
| 1B | Cover 1B; receive throw |
| 2B | Cover 2B |
| SS | Cover 3B (if 3B fields); otherwise cover 2B |
| 3B | Field if to 3B; look at runner on 3B; throw to 1B or home |
| LF | Back up 3B |
| CF | Back up 2B |
| RF | Back up 1B |

---

## Section 16: Fly Balls — Additional Runner Combinations

### 16.1 Fly Ball — Runner on 3B Only (Tag-Up)

Similar to Section 3, but only the run is at stake — no complication of multiple runners.

| Position | Responsibility |
|----------|----------------|
| P | Back up home plate |
| C | Set up at home; prepare for tag play |
| 1B | Cover 1B |
| 2B | Cover 2B |
| SS | Move into relay between OF and home (based on field — see 3.1–3.3) |
| 3B | Cover 3B; remind runner of tag-up timing; do not let runner leave early |
| LF | Back up (if not catching) |
| CF | Back up (if not catching) |
| RF | Back up (if not catching) |

**Outfielder's throw:** Hard throw toward the relay/cutoff aligned to home. Catcher directs cut-off: "Cut!" or "Let it go!"

**Shallow fly ball with runner on 3B:** If the fly ball is shallow and catchable, the runner may still tag and score — outfielder must get the ball back in **immediately**. If the outfielder is unsure they can catch it, they must weigh: catch and give up run vs. play it off the hop and keep runner at 3B.

---

### 16.2 Fly Ball — Runners on 1B & 3B

Runner on 3B will tag and try to score. Runner on 1B may tag and advance or hold.

| Position | Responsibility |
|----------|----------------|
| P | Back up home |
| C | Set up at home |
| 1B | Cover 1B |
| 2B | Cover 2B |
| SS | Relay between OF and home (LF/CF side) or cover 2B (RF side) |
| 3B | Cover 3B; watch runner on 3B for early tag-up |
| LF | Back up if not catching; catch and throw toward relay to home |
| CF | Back up if not catching; catch and throw toward relay to home |
| RF | Back up if not catching; 1B serves as relay to home (see 3.3) |

**Primary target:** Throw home — the runner on 3B is the lead runner and is trying to score.

---

### 16.3 Fly Ball — Bases Loaded

All runners may tag and advance. Focus is on the runner scoring from 3B.

| Position | Responsibility |
|----------|----------------|
| P | Back up home |
| C | Set up at home |
| 1B | Cover 1B (or relay if fly to RF) |
| 2B | Cover 2B (or relay if fly to RF) |
| SS | Relay to home on LF/CF side; cover 2B on RF side |
| 3B | Cover 3B |
| LF | Catch; throw to relay toward home |
| CF | Catch; throw to relay toward home |
| RF | Catch; throw to 1B/2B relay toward home |

**With fewer than 2 outs:** After the catch, runners may tag up from all bases — 3B runner scores, 2B runner advances to 3B, 1B runner advances to 2B. The play at home is the priority.

**With 2 outs:** Runners run on contact. If caught, inning is over; no play needed.

---

## Section 17: Pick-Off Plays

### 17.1 Pick-Off at First Base

| Position | Responsibility |
|----------|----------------|
| P | From the stretch: step off and throw to 1B; or use a pick-off move (left-handed pitcher only from the rubber) |
| C | Alert; can signal the pick-off to P; back up home |
| 1B | Set up with right foot on inside corner of bag; receive throw; apply tag quickly to the inside of the bag |
| 2B | Cover 2B in case runner retreats and tries to advance |
| SS | Back up 2B |
| 3B | Hold near 3B |
| LF | Back up |
| CF | Back up |
| RF | Back up |

> ⚠️ **COACH DECISION — Pick-off sign at 1B:** How does P know to throw?
> - **Pitcher's own read:** P picks his spot when he thinks the runner has a big lead
> - **Catcher signal:** C puts down a sign (e.g., two fingers pointing down = pick at 1B)
> - **1B signal:** 1B taps glove or gives a verbal indicator to P
>
> Also define: does 1B give a "step" (fake) to hold runner, or only move when P commits to throw?

---

### 17.2 Pick-Off at Second Base

More complex — requires SS or 2B to break to the bag on a count or visual cue.

| Position | Responsibility |
|----------|----------------|
| P | From the stretch: wheels and throws to 2B when SS/2B breaks; must see the player break before throwing |
| C | Can initiate timing play with P; back up home |
| 1B | Hold near 1B |
| 2B | Break to 2B bag on the count or cue to receive throw (alternates with SS) |
| SS | Break to 2B bag on the count or cue to receive throw (alternates with 2B) |
| 3B | Stay near 3B |
| LF | Back up 2B |
| CF | Back up 2B |
| RF | Back up 2B |

> ⚠️ **COACH DECISION — Pick-off play at 2B:**
> - **Timing play:** SS (or 2B) breaks to 2B after a set number of pitches or a set count; P wheels and throws on a cue (e.g., glove tap by C)
> - **Daylight play:** P looks at runner; when SS gets between P and runner ("daylight"), P throws immediately — requires P and SS to practice reading each other
> - **Snap throw:** C receives pitch and quickly snaps throw to 2B; SS/2B must already be breaking
>
> Pick-offs at 2B require repetition in practice. Do not use in a game without drilling it first.

---

### 17.3 Pick-Off at Third Base

| Position | Responsibility |
|----------|----------------|
| P | Step off and throw to 3B when 3B signals or P reads a big lead |
| C | Can signal the play; will cover home |
| 1B | Cover 1B |
| 2B | Cover 2B |
| SS | Back up 3B in case of wild throw |
| 3B | Signal P (e.g., verbal "now!" or glove tap); set up to receive throw; apply quick tag on inside of bag |
| LF | Back up 3B |
| CF | Back up |
| RF | Back up |

> ⚠️ **COACH DECISION — Pick-off sign at 3B:** Same options as 1B: pitcher's read, catcher signal, or 3B initiates. 3B must be careful not to telegraph the play by moving too early.

---

## Section 18: Wild Pitch / Passed Ball — Runner on 2B

### 18.1 Wild Pitch or Passed Ball — Runner on 2B

Runner on 2B will attempt to advance to 3B (and possibly home on a ball that gets far away).

| Position | Responsibility |
|----------|----------------|
| P | Back up 3B |
| C | Chase the ball immediately; throw to 3B |
| 1B | Cover 1B |
| 2B | Cover 2B |
| SS | Cover 3B; receive throw |
| 3B | Sprint toward 3B bag; or cover if not far from it |
| LF | Back up 3B |
| CF | Back up 3B |
| RF | Back up |

**If the ball gets very far past C:** Runner may try to score. C must decide quickly whether to throw to 3B (likely out) or throw to P covering home.
